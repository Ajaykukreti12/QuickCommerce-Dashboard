# Quick Commerce MIS Dashboard (Excel + Power BI)

This project demonstrates an end-to-end data analysis workflow for a Quick Commerce business, starting from raw Excel data preparation to an interactive Power BI dashboard.

The goal of this project was to transform raw operational data into meaningful business insights using data cleaning, transformation, and visualization techniques.

---

# Project Workflow

Raw Data → Power Query (Cleaning & Transformation) → Excel Master Dataset → Power BI Dashboard → Business Insights

---

# Dataset

The dataset is stored in Excel format and contains order-level transactional data.

Main fields include:

• Order ID  
• Order Date  
• Product Category  
• Customer / Location Details  
• Quantity  
• Revenue / Sales  
• Delivery SLA & Delivery Time  

This dataset was cleaned and structured before building the dashboard.

# Data Preparation (Excel + Power Query)

The raw dataset was processed using Power Query to create a structured master dataset.

Steps performed:

• Removing null and inconsistent values  
• Changing data types  
• Formatting date columns  
• Mapping codes to category names  
• Creating calculated columns  
• Preparing a clean master dataset for analysis  

This automated process allows the dataset to be refreshed easily.

---

# Dashboard Features (Power BI)

The Power BI dashboard provides interactive insights into business performance.

Key dashboard components include:

### KPI Cards
• Total Orders  
• Total Revenue  
• Total Quantity  
• Average Order Value  

### Visualizations
• Category wise sales (Bar Chart)  
• Monthly order trend (Line Chart)  
• Category distribution (Donut Chart)  
• Detailed order table  

### Interactive Filters
Users can filter the dashboard using slicers such as:

• Date  
• Category  
• City  
• Product  

### Reset Filter Button
A bookmark-based reset button was implemented to quickly return the dashboard to its default view.
